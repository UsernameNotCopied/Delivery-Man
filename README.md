## 2. ΠΕΡΙΓΡΑΦΗ ΛΟΓΙΣΜΙΚΟΥ

Το "Delivery Man" είναι ένα παιχνίδι διανομής σε δισδιάστατο (2D) περιβάλλον. Ο χρήστης ελέγχει έναν οδηγό σε μια πόλη με ενεργή κυκλοφορία οχημάτων. Σκοπός είναι η διαχείριση παραγγελιών από φαρμακεία προς πελάτες, με στόχο τη μεγιστοποίηση του κέρδους.

  

**Κύρια Χαρακτηριστικά:**

*  **Σύστημα Κυκλοφορίας:** Αυτόνομα οχήματα που κινούνται στους δρόμους, αλλάζουν πορεία στις διασταυρώσεις, σέβονται τις λωρίδες κυκλοφορίας και αποφεύγουν συγκρούσεις.

*  **Σύστημα Πλοήγησης:** Υπολογισμός της βέλτιστης διαδρομής προς τον στόχο και οπτική απεικόνισή της στον χάρτη.

*  **Οικονομικό Μοντέλο:** Οι παραγγελίες έχουν αμοιβή που μειώνεται με την πάροδο του χρόνου. Τα ατυχήματα επιφέρουν χρηματικές ποινές.

*  **Δυναμική Δυσκολία:** Το παιχνίδι δυσκολεύει προοδευτικά με κάθε 3 ολοκληρωμένες παραγγελίες. Η κίνηση αυξάνεται, η μείωση της αμοιβής επιταχύνεται και οι ποινές διπλασιάζονται.

## 4. ΟΔΗΓΙΕΣ ΕΓΚΑΤΑΣΤΑΣΗΣ/ΜΕΤΑΓΛΩΤΤΙΣΗΣ
Για τη μεταγλώττιση του πηγαίου κώδικα σε εκτελέσιμο αρχείο, ακολουθήστε τα παρακάτω βήματα.

1. Εγκαταστήστε το πακέτο Raylib (συνήθως στο `C:\raylib`).
2. Ανοίξτε το τερματικό **w64devkit.exe** (βρίσκεται στο `C:\raylib\w64devkit`).
3. Πηγαίνετε στη τοποθεσία του παιχνιδιού (π.χ. `cd C:/Users/YourName/Desktop/DeliveryMan`).
4. Εκτελέστε την εξής εντολή:

```bash
gcc src/main.c src/map.c src/player.c src/order.c src/pathfinding.c src/traffic.c -o DeliveryMan.exe -O2 -Wall -I C:/raylib/raylib/src -L C:/raylib/raylib/src -lraylib -lopengl32 -lgdi32 -lwinmm
```

## 6. ΕΓΧΕΙΡΙΔΙΟ ΧΡΗΣΗΣ

  

### Σκοπός

Ο στόχος είναι να ολοκληρώσετε όσες περισσότερες παραδόσεις γίνεται. Αν τα χρήματά σας φτάσουν στο μηδέν, το παιχνίδι τελειώνει (Game Over).

  

### Χειρισμός (Πληκτρολόγιο)

|  **W / A / S / D**  |  Κίνηση του οχήματος (Πάνω, Αριστερά, Κάτω, Δεξιά). |

|  **P**  | Αποδοχή νέας παραγγελίας. |

|  **1 / 2 / 3**  |  Επιλογή ενεργής παραγγελίας. Ενεργοποιεί το GPS για τη συγκεκριμένη παραγγελία. |

|  **ESC**  |  Έξοδος από την εφαρμογή. |

  

### Μηχανισμοί Παιχνιδιού

*  **Ροή Παραγγελίας:** Υπάρχουν έως 3 διαθέσιμες παραγγελίες στην οθόνη. Επιλέγετε μία (π.χ. πατώντας '1'), την αποδέχεστε ('P'), οδηγείτε στο Φαρμακείο (πορτοκαλί ένδειξη) και την παραδίδετε στον Πελάτη (κόκκινη ένδειξη).

*  **GPS:** H μπλε γραμμή δείχνει τη συντομότερη διαδρομή προς τον τρέχοντα στόχο.

*  **Level Up:** Κάθε 3 ολοκληρωμένες παραδόσεις, το επίπεδο δυσκολίας αυξάνεται:

* Η κυκλοφορία αυξάνεται κατά 10 αυτοκίνητα.

* Η ποινή σύγκρουσης διπλασιάζεται.

* Ο ρυθμός μείωσης της αμοιβής αυξάνεται.

*  **Game Over:** Όταν το ποσό χρημάτων γίνει 0, το παιχνίδι τελειώνει.
